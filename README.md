# AI-Dynamic-Pathfinding-Agent

A pygame-based visualizer for comparing A* and Greedy Best-First 
Search (GBFS) on a customizable grid with dynamic obstacle spawning.

## Requirements
- Python 3.8+
- pygame

## Installation
pip install pygame

## How to Run
python main.py

## Controls
| Input          | Action                  |
|----------------|-------------------------|
| Left Click     | Place / Remove Wall     |
| Right Click    | Move Start Node         |
| Middle Click   | Move Goal Node          |

## Features
- A* Search (optimal, uses g + h)
- Greedy BFS (fast, uses h only)
- Manhattan and Euclidean heuristics
- Dynamic obstacle spawning with replanning
- Real-time metrics: nodes visited, path cost, time (ms)
